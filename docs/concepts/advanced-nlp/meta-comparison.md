# Milton and Meta Comparison

| ***Name of Violation*** | ***Milton Model*** | ***Meta Model*** |
| --- | --- | --- |
| **GENERALISATION** |  |  |
| Universal Quantifier (All, Every, None..) | *"You're always able to find new ways”*
*"You will never be lost of ideas”* | "You never talk to me"
"I'm always late." |
| Modal Operator of Necessity | *"You must already have the answer”*
*"Taking this to where it has to go* | "I mustn't go yet"
"I have to phone my mother" |
| Modal Operator of Possibility | *“You can know...”*
*"That you will be able to discover.”* | "I can't speak to my boss"
"There's no way I'll do that" |
| Presuppositions | *"This will be easy for someone of your ability..."* | “Many have tried before... and failed.” |
|  |  |  |
| **DELETION** |  |  |
| Simple Deletion | *"You are listening...”* | "I feel depressed."
"I don't know.” |
| Comparative Deletion | *"You are going deeper.... feeling more relaxed..."* | "It's getting worse" |
| Nominalisation | *"You are making discoveries, finding new insights...”*
*"As you gain a new understanding.”* | "I want to make a good impression.”
“Our relationship is changing.” |
| Lack of Referential Index | *"Remembering those people who have been important to you.”*
*"People are good at this."* | "They came too late"
“People don’t like me.” |
| Unspecified Verb | *"Making sense in your own way.”*
*"Knowing that you know...”* | "He hurt me"
"I noticed it was missing" |
|  |  |  |
| **DISTORTION** |  |  |
| Complex Equivalence | *"That means that you are finding what you need to do..."*
*"Sitting quietly means you are going inside..."* | "When you hold me like that I know you love me."
"With that smile she must have lots of friends" |
| Cause & Effect | *"As you listen to my voice you will go deeper into trance..."*
*"Each breath will make you relax even more..."* | "Her crying makes me sad"
"His love makes me want to go on.” |
| Mind Reading | *"I know that you are getting curious...”*
*"Your unconscious mind is able to make sense of this..."* | "You're annoyed with me"
"You're not going to like this" |
| Lost Performative | *"It's important that you go back to your first memory."*
*"And it's a good thing to notice...”* | "It's fun to be free"
"Big boys don't cry" |
